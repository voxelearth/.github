## Who We Are

We build **Voxel Earth** — a pipeline that voxelizes real-world 3D data (e.g., Google’s Photorealistic 3D Tiles) into interactive worlds for **Minecraft**, **VR**, and the **web**. We’re actively developing open components and welcome contributions.
